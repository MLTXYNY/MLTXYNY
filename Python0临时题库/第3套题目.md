## 任务1：
输入三个整数 x、y、z，请把这三个数由小到大输出。
要求：使用分支结构语句实现。
```python
# _*_ coding:utf-8 _*_
print("本代码实现：输入三个整数 x、y、z，把这三个数由小到大输出。")
x,y,z=map(int,input("请输入三个整数 x、y、z（注意：用逗号分隔）:").split(','))
#print(x,y,z)
if x>y:   #将x,y中较大的值赋给y
    x,y=y,x
if y>z:   #将y,z中较大的值赋给z
    y,z=z,y
if x>y:    #将最小的值赋给x
    x,y=y,x
print(f"你输入的这三个数按从小到大的顺序输出依次为：{x}、{y}、{z}")
```
